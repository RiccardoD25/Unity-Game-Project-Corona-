<h1>Unity C# Scripts - Game Development</h1>

<h2>Description</h2>
This project is called Corona and it consists of simple C# scripts that walks the user through each phase of it's job function. The source code allows you to run the game from intro screen using the left and right controllers as wel as to shoot injectors at the targets in order to vanquish each and every one of them until there are none left to destroy. There are multiple scripts from begining to end which indicates each phase of the game. This is a VR (Virtual Reality) game as i did build it to the Quest 2 as intended. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Visual Studio 2019</b> 
- <b>Unity 3D</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
